# 数据处理流程

本页用于描述一次典型的数据处理流程。后续会配合软件截图补充每一步的具体位置和操作细节。

## 典型流程

1. 创建或打开项目。
2. 导入实验数据文件。
3. 检查数据表和样品结构。
4. 选择需要处理的数据范围。
5. 应用数据处理操作。
6. 在绘图工作区检查结果。
7. 调整图表样式。
8. 导出结果、图谱或项目文件。

## 使用建议

- 每次开始处理新实验前，建议先保存一个项目文件。
- 对需要重复使用的处理步骤，优先考虑保存为模板。
- 对关键结果，建议同时保存项目文件和导出结果，便于后续追溯。
