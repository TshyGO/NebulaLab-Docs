# 数据准备

## 何时用这一页

数据准备是工作区的第一步。这一步做三件事：把仪器导出的 CSV、TXT、Excel 文件导入 Nebula Lab，整理成「分组 → 样品」的结构，再确认每个表格解析正确。

数据整理好，才能进入下一步绘图和处理。

## 基本步骤

1. 在左侧「原始数据管理」面板点「导入文件」，或直接把文件拖进这个面板。支持 csv、txt、xlsx、xls、dat、xy、asc 格式。
2. 在「导入文件」窗口里设置「导入分组」，检查「基本解析」，看「轻量预览」里的前 5 行是否正确。
3. 点「确认导入」。
4. 在左侧样品树里检查导入后的「分组」和「样品」结构。
5. 在中间「表格工作区」用「当前样品」逐个核对每个样品的行和列。

## 关键选项

### 导入分组

「导入文件」窗口里可以选择导入方式：

- 本次一起导入为一组：多个文件合成一个分组。
- 每个文件单独一组：每个文件各自成组。

分组名可以留空。留空时，一起导入会自动命名，单独导入会直接用文件名。

### 基本解析与表头

导入时会自动识别表头结构，「使用表头」「表头行」「跳过行数」一般不用动。只有预览里表头明显不对时，再手动调「表头行」。

导入 Excel 文件时，「分隔符」和「编码」不可用，这是正常的。

### Excel 的 Sheet

导入 Excel 时，「预览 Sheet」可以切换要预览的工作表。如果文件有多个 Sheet，「导入范围」可以选「全部 Sheet」或「当前 Sheet」。

### 样品树管理

在样品树里可以：

- 创建分组、重命名分组或样品、创建副本。
- 拖动样品到另一个分组，移动它。
- 把样品拖到空白处，新建一个分组。

删除文件会连带删除它下面的全部样品和分组，删除分组会连带删除组内样品。删除前弹窗会提示。

### 数据表查看

中间「表格工作区」用「当前样品」下拉框切换查看的样品。导入的 Excel 会在表格上方显示 Sheet 标签页。表格里的单元格可以直接编辑。

## 常见坑

- 表头识别错了：在「导入文件」窗口里手动调「表头行」，对照「轻量预览」确认。
- 格式不支持：只接受 csv、txt、xlsx、xls、dat、xy、asc，其它格式先在原软件里转一下。
- 编辑作用域要小心：当你选中的是整个「分组」时，改一列会作用到组内所有样品。只想改一个样品，先在「当前样品」里单独选中它再改。
