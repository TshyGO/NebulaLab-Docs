# 用户手册

这份手册会从使用者视角介绍 Nebula Lab：软件能做什么、每个页面解决什么问题，以及如何完成一次完整的数据处理流程。

## 建议阅读顺序

1. 先阅读[核心概念](./concepts.md)，了解项目、样品、工作流和模板之间的关系。
2. 按照[数据处理流程](./workflow.md)完成一次基础操作。
3. 如果需要复用或分享处理流程，再阅读[社区与模板](./community.md)。

## 当前版本重点

当前版本主要围绕三个方向：

- 实验数据导入与整理。
- 数据处理、绘图和导出。
- 工作流模板、插件和社区资源。

后续版本会逐步加入配方计算、实验记录、本地实验库和课题组自部署服务器等能力。
