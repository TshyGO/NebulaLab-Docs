# About Pro Edition

In order to support the sustainable healthy development and iteration of NebulaLab, we have launched the **Pro Edition** for advanced researchers, high-confidentiality research groups, and industrial R&D teams.

The following are the differences in features and services between the Standard edition and the Pro edition:

---

## 1. Feature Comparison Table

| Feature Module | Standard Edition | Pro Edition |
| :--- | :--- | :--- |
| **Basic Data Prep** | Supports common CSV/TXT/Excel, manual sample organization | Supports million-row level **fast file streaming import**, unlimited sample management |
| **Advanced Data Algorithms** | Built-in basic cropping, scale zooming, and other common preprocessing | **Advanced multi-peak fitting (Gauss/Lorentz), intelligent overlapping peak parsing, adaptive multi-algorithm peak finding** |
| **Workflow Efficiency** | Manually apply processing templates to a single local sample | **Multi-core CPU parallel computing, automatic batch import and streaming data cleaning** |
| **Private Deployment & Ecosystem** | Free installation of public community plugins and themes | **Supports enterprise/research group internal private plugin market, LAN secure package distribution and management** |
| **Cloud Collaboration & Sharing** | Runs standalone, project files must be manually sent and circulated | **Experimental data private cloud synchronization, one-click secure sharing of experiment status within the research group** |
| **Technical Support & Customization**| Asynchronous replies via community GitHub Issues | **24-hour dedicated algorithm expert online support, customization and integration of specific instrument parsing algorithms** |

## 2. How to Activate Pro Edition

Activating the Pro edition is simple. After purchasing the authorization code, it can be unlocked seamlessly without reinstalling the software.

1. **Obtain Authorization Code**: Please visit the official pricing page `https://nebulalab.io/pricing` to purchase the authorization code *(simulated service page to protect privacy)*.
2. **Fill in Activation Code**: Start the software, go to **"Settings" → "License Management"**.
3. **One-Click Activation**: Paste the activation code you obtained into the input box and click the **"Activate"** button. The system will automatically connect to the secure authorization server for online verification.
4. **Offline Self-Deployed Activation**: For key university laboratories or military research institutes with extremely high security levels and physical network disconnection, we provide a **local LAN authorization server deployment solution**. Please contact commercial customer service and technical support directly to obtain exclusive packages and offline key files.

## 3. Privacy Security Commitment

> [!CAUTION]
> **Local First & Scientific Research Privacy Security**
> Whether it is the Standard edition or the Pro edition, NebulaLab adheres to the principle of **"absolute isolation of local data"**. All raw data reading, smoothing filtering, and plotting calculations of experimental instruments **take place entirely on your local computer**. Unless you actively click cloud sharing, absolutely no data will be secretly uploaded to external servers. We deeply understand the privacy and cutting-edge value of experimental data to scientific researchers.
