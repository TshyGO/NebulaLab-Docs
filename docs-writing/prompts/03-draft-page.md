# Prompt: Draft Manual Page

You are writing a first draft of one Nebula Lab user manual page.

## Writing Rules

- Use natural Chinese.
- Write for researchers, not developers.
- Do not invent features.
- Do not use generic AI or marketing language.
- If a detail is missing, write `需要确认：...`.
- Keep paragraphs short.
- Every step should correspond to something the user can do in the app.

## Inputs

- Page brief: `{{page_brief}}`
- Fact pack: `{{fact_pack}}`
- Style guide: `{{style_guide}}`
- Terminology: `{{terminology}}`

## Output

Write the Markdown page only.
