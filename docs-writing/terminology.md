# Terminology

Use these terms consistently.

| Term | Use | Avoid |
| --- | --- | --- |
| Nebula Lab | Product name | Nebula, Nebula Graph unless referring to old release text |
| 数据准备 | The first main workspace step | 数据导入页, 预处理模块 |
| 绘图工作台 | The second main workspace step | 分析页, 作图模块 |
| 导出 | The third main workspace step | 输出模块 |
| 工程 | `.nbp` project file | 项目文件 unless quoting UI |
| 样品 | A loaded data item that can be selected, processed, and plotted | 曲线 when referring to the whole item |
| 分组 | Group-level organization of samples | 文件夹 unless literally describing folders |
| 导入配置 | Parsing and import settings | 导入模板, 读取规则 |
| 工作流 / 流程 | Reusable processing sequence | 自动化闭环 |
| Origin 导出 | Sending data/plot work to Origin | Origin 同步 unless style sync is specifically described |
| 社区 | Built-in community resources area | 市场 unless the UI says market |
| 插件 | Local or community extension | 扩展程序 unless the UI says so |

When the UI text changes, update this file before rewriting pages.
